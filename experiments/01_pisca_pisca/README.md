# Experimento 01 — Pisca-pisca com LEDs

## Objetivos
1. **Intermediário:** Validar a configuração de um pino de saída e fazer o LED Azul (D12) piscar.
2. **Final:** Implementar uma alternância rítmica entre o LED Azul (D12) e o Vermelho (D13), simulando um sinal de alerta.

## Componentes
- LED Azul (GPIO 12)
- LED Vermelho (GPIO 13)

## Entregáveis
- `template.py` completado com as duas etapas de temporização.
- Demonstração dos LEDs alternando a cada 500ms.
