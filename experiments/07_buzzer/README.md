# Experimento 07 — Buzzer e Frequência

## Objetivos
1. **Intermediário:** Entender o uso de PWM para geração de áudio e validar o funcionamento do Buzzer (GPIO 5).
2. **Final:** Criar uma interface sonora onde cada botão (SW1 e SW2) gera uma nota musical diferente.

## Componentes
- Buzzer Piezoelétrico (GPIO 5)
- Botões SW1 e SW2

## Entregáveis
- Código com função `beep` ou controle de `freq`.
- Demonstração de dois tons distintos.
