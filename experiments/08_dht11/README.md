# Experimento 08 — Temperatura e Umidade com DHT11

## Objetivos
1. **Intermediário:** Aprender a utilizar bibliotecas específicas (`dht`) para sensores digitais e validar a leitura.
2. **Final:** Criar um sistema de monitoramento de conforto térmico que alerta visualmente se a umidade estiver fora dos padrões ideais.

## Componentes
- Sensor DHT11 (GPIO 4)
- LED Vermelho (GPIO 13)

## Entregáveis
- Leitura estável exibida no Shell a cada 2 segundos.
- Lógica de alerta condicional baseada na umidade.
